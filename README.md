# HTML-SCSS

Le but de ce projet est de recréer un template HTML tout en utilisant un compilateur CSS prénommé SCSS. 

Les techniques utilisées durant ce projet sont les suivantes : 
 - Les selecteurs CSS 
 - Les flexibles 
 - les inlines-blocks
 - les girds 
 - les mixins 
 - le compilateur SaSS
 - Responsive Design
